네트워크 연결과 구성요소

네트워크 연결 구분

네트워크는 규모에 따라 LAN,MAN,WAN 3가지로 구분된다.

1.LAN(Local Area Network) : 사용자 내부 네트워크 (근거리 통신망으로 주로 집 , 사무실 단위)
2.MAN(Metro Area Network) : 한 도시 정도를 연결하고 관리하는 네트워크
3.WAN(Wide Area Network) : 멀리 떨어진 LAN을 연결해주는 네트워크

네트워크 회선

인터넷 회선

  • 인터넷 접속을 위해 통신사업자와 연결하는 회선

인터넷 가입자와 통신사업자간에는 직접 연결되는 구조가 아니라 전송 선로 공유 기술을 사용한다. 예를 들어 아파트 광랜은 아파트에서 통신사업자까지 연결한 회선을 아파트 가입자가 공유하는 구조임. 즉, 전송 선로를 공유하므로 전용 회선과 다르게 속도가 보장되지 않는다 (주변 사용량에 따라 속도가 느려질 수도 있다. )

인터넷 회선 종류는 다음과 같다.

  • 광랜(이더넷) , FTFH , 동축 케이블 인터넷 , xDSL

전용 회선

  • 가입자와 통신사업자 간에 대역폭(bandwith,링크 용량)을 보장해주는 서비스.

가입자와 통신사업자 간에는 전용 케이블로 연결되어 있고, 통신사업자 내부에서 TDM(Time Division Mulitiplexing) 같은 기술로 마치 직접 연결한 것처럼 통신 품질을 보장해준다.

인터넷 전용 회선

  • 인터넷 회선에 대한 통신 대역폭을 보장해주는 서비스

가입자가 통신사업자와 연결되고 이 연결이 다시 인터넷과 연결되는 구조이다.

VPN (Virtual private network , 가설 사설망 )

  • 물리적으로는 전용선이 아니지만, 가상으로 직접 연결한 것 같은 효과가 나도록 만들어주는 네트워크 기술

ex) 재택근무중인데 회사 사설망에 연결되어있는 것처럼 업무 가능

  • 통신사업자 VPN : VPN 서비스를 ISP에서 제공하며 전용선의 거리에 비례하는 비용문제가 있다.
  • 가입자 VPN : 일반 인터넷망을 이용해 가상 네트워크를 구성할 수 있다.

네트워크 구성 요소

네트워크 인터페이스 카드 (NIC)

흔히 랜카드,네트워크 카드, 네트워크 인터페이스 컨트롤러로도 불리는 네트워크 인터페이스 카드는 컴퓨터를 네트워크에 연결하기 위한 하드웨어 장치이다.

노트북과 데스크탑 PC에서는 기본적으로 온보드 형태로 장착됨으로 별도로 추가할 필요는 없으나, 서버의 경우에 여러 네트워크에 동시에 연결하거나 혹은 더 높은 대역폭이 필요한 경우, 네트워크 인터페이스 카드를 추가로 장착한다.

네트워크 인터페이스 카드의 주요 역할은 다음과 같다.

  • 직렬화 (Serialization) : 전기적 신호를 bit열의 데이터 신호 형태로 , bit열의 데이터 신호 형태를 전기적 신호 형태로 변환시켜주는데, 이러한 상호 변환작업을 직렬화라고 한다.

  • MAC 주소 : 네트워크 인터페이스 카드는 고유한 MAC주소를 가지고 있어서, 받은 패킷의 도착지 MAC주소가 자신의 MAC주소가 아니면 폐기하고 맞으면 전달한다.

  • 흐름 제어 (Flow control) : 이미 통신 중인 데이터 처리 떄문에 새로운 데이터를 받지 못하는 상황에서 데이터 유실 방지를 위해 상대방에서 통신 중지를 요청할 수 있다.

케이블

케이블의 종류는 다음과 같다.

  • 트위스트 페어 케이블
  • 동축 케이블
  • 광케이블

현재 물리계층에서 가장 많이 사용되는 네트워크 기술은 이더넷으로 이더넷에서 사용하고 있는 케이블과 커넥터를 정리하려고 한다.

현재 대중화된 이더넷 표준은 기가비트 이더넷과 10기가비트 이더넷이다.
이더넷에서도 케이블의 종류, 인코더의 종류 등으로 세분화해서 여러 가지 표준으로 나뉘지만 대중적으로 많이 사용되는 표준은 3가지이다.

  1. 1,000BASE-T/10GBASE-T

트위스트 페어 케이블을 이용하는 기가 이더넷 표준

  1. 1,000BASE-SX/10GBASE-SR

멀티모드 광 케이블을 사용하고 비교적 짧은 거리를 보낼 수 있는 이더넷 표준

  1. 1,000BASE-LX/10GBASE-LR

싱글모드 광 케이블을 사용하고, 비교적 긴 거리를 보낼 수 있는 이더넷 표준

각 1,000BASE-T 이더넷 표준에서 명칭의 의미는 다음과 같다

1,000 : 1,000Mbps 속도 / BASE : 채널 종류 (BASE : 단일 채널 , BROAD : 다채널) / T : 케이블 타입 (T : 트위스트 페어 케이블)

케이블 구조

케이블은 물리적으로 케이블 본체 ,커넥터, 트랜시버와 같은 여러 요소로 나뉜다.

트위스트 페어 케이블

트위스트 페어 케이블은 쉴드를 장착한 STP/FTP 케이블 , 쉴드가 없는 UTP 케이블로 나뉜다.

RJ-45 커넥터를 이용하고, 케이블 본체와 함께 연결되어 분리할 수 없는데, 컴퓨터에 있는 랜포트에 끼우면 네트워크에 연결된다.

카테고리 단위로 케이블 등급을 나눌 수 있는데, 가장 일반적으로 많이 사용되는 케이블은 카테고리 5E케이블이나 데이터센터와 같이 높은 대역폭을 지원해야할때는 10G 이상 을 사용한다고 한다.

동축 케이블

과거에 LAN에서도 사용되었지만, 고가로 케이블 TV나 인터넷 연결을 위해서만 사용되어 왔다.

광케이블

다른 구리선(UTP)보다 신뢰도가 높으며 , 먼거리 까지 통신 가능함으로 높은 대역폭을 요구하거나 먼 거리를 통신해야하는 네트워크 장비간에 주로 사용됨 (ex) 라우터간 , 스위치간 )
싱글모드,멀티모드 2가지로 나뉘는데, 싱글모드는 케이블 굵기가 가늘며, 신호를 보내는 광원으로 레이저를 사용하는 반면 멀티모드는 굵은 케이블을 사용하며 광원으로 LED를 사용한다.

넓은 광 전송로에 여러가지 광원이 전송됨으로 멀티모드, 하나의 레이저 신호가 가느다란 전송로를 통과함으로 싱글모드라고 하고, 반사각도가 작은 레이저 광원을 사용하는 싱글모드가 휠씬 먼거리를 전송할 수 있기 떄문에 비교적 멀티모드보다 고가이다.

커넥터

케이블의 끝부분으로 네트워크 장비나 네트워크 카드 (네트워크 인터페이스 카드)에 연결되는 부분

트랜시버

다양한 외부 신호를 컴퓨터 내부의 전기 신호로 바꾸어주는 역할을 함

트랜시버를 사용하지 않는다면 케이블에 맞는 길이나 속도마다 다른 네트워크 인터페이스 카드를 별도로 구비해야 하지만, 트랜시버만 변경하면 통신 길이와 속도를 조절할 수 있다.

최근 생산되는 네트워크 인터페이스 카드는 트랜시버를 대부분 지원하고 있다.

허브

케이블과 동일하게 1계층에서 동작하는 장비로 여러 대의 장비를 연결할 목적으로 사용되는데, 단순히 들어온 전기신호를 모든 포트로 내보내 접속된 모든 단말이 경쟁하게 됨으로 전체 네트워크 성능이 줄어드는 단점이 있다.

스위치

허브와 동일하게 여러 장비를 연결하는 기능에 추가적으로 통신을 중재하는 2계층에서 동작하는 장비이다. 허브와 스위치의 내부 동작 방식은 다르지만 , 여러 장비를 연결하고 케이블을 한곳으로 모아주는 역할은 같으므로 허브라는 용어를 공통적으로 사용한다.

  • 1계층 장비인 허브와의 차이점 : 허브는 전기 신호를 재생성해 출발지를 제외한 모든 포트에 전기 신호를 보내는 역할이지만, 스위치는 허브와 다르게 MAC 주소를 이해할 수 있어서 목적지의 MAC 주소를 보고, 해당 포트로만 전기 신호를 보낸다.

라우터

3계층에서 동작하는 네트워크 장비로 라우터는 라우터 내부의 라우팅 테이블을 보고 최적의 경로로 패킷을 포워딩한다.

로드 밸런서

4계층에서 동작하는 네트워크 장비로 , application 계층 (7계층) 에서 application protocol 의 특징을 이해하고 동작하는 로드 밸런서는 별도로 ADC (Application Delivery Controller)라고 부른다.

로드 밸런서는 4계층 포트 주소를 확인하는 동시에 IP주소를 변경할 수 있다. 웹에서 웹 서버를 증설할때, 로드 밸런서를 웹 서버 앞에 두고 웹 서버를 늘려 주면, 대표 IP는 로드밸런서가 가지고 각 웹 서버로 패킷의 목적지 IP 주소를 변경해준다. 따라서 여러대의 웹 서버로 워크로드를 분산시키고, 일부 서버가 다운되더라도 서비스가 복구되도록 도와준다.

방화벽

4계층에서 동작하는 보안 장비로, 방화벽을 통과하는 패킷의 3,4계층 정보를 확인해서 정책과 비교해 패킷을 버리거나 포워딩해준다.

공유기 , 모뎀

일반적으로 가정에서 사용하는 공유기는 2계층의 스위치, 3계층의 라우터 , 4계층의 NAT기능을 모두 한곳에 모아놓은 장비이다. 모뎀은 짧은 거리를 통신하는 기술과 먼 거리를 통신할 수 있는 기술이 달라, 이를 변환해주는 장비이다.
